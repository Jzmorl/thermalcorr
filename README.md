thermalcorr
===========

A collection of thermohydraulic correlations for thermal systems and heat exchangers

Hierarchy:

* Internal Flow
  * Heat transfer and pressure drop
    * Tubes/non-circular ducts
    * Microchannels
* External Flow
  * Heat transfer and pressure drop
    * Fins
    * 

Developers
==========
